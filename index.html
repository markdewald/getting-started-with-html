What did you learn?!?!:

  1. How the Web Works: Web Hyperlinks, HTML, URL, HTTP, etc.:
    • HTTP Requests sent to a server
    • Server finds the appropriate HTML document and sends it back to the user's computer where a web browser interprets the page and displays it on the user's screen.
  2. HTML stands for Hypertext Markup Language:
    • Text content which describes "what you see"
    • Markup which describes "how it looks"
  3. Tags > Elements > Attributes
    • HTML documents are made up by HTML elements
    • Detailed description of HTML elements can be found here:
    • Browsers tell the type of each element by using HTML tags
  4. Inline vs Block Elements
    • Span - defines a section in a document (inline)
    • Inline examples: br, b, em, img, span
    • Div - defines a section in a document (block-level)
    • Block level examples: p, div
